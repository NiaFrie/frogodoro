**Titel:** Frogodoro

**Beschreibung:** Frogodoro ist eine PWA die als Lernunterstützung für jegliche Themenfelder (z.B. Studium) dient. Der Frosch-Lernpal leistet Gesellschaft während der Timer läuft und die App speichert anschließend die Lernzeiten, wodurch die eigene Zeitinvestition über längere Zeit getracked werden kann.
